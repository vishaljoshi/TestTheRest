Yet another testing framework?

Yes! but this one is built from ground up to test JSON based restful services on its native platform . Itâ€™s easy to write/edit and play the test cases making test case management piece of cake. We are continuously improving it in the background.

Who can use it?

Any one from front end developers to middle ware/backend guys . As long as you are building web service over HTTP and using JSON you should be good.

Does it provide reporting?

Yes, we do support a CSV report file which provides test case result as well as response time.
